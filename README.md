# Sample Python Application

A simple **task manager** (to-do list) built in Python to demonstrate basic application structure, functions, and unit testing.

## Features

- Add tasks
- Mark tasks as complete
- List all tasks with their status

## Requirements

- Python 3.8+
- `pytest` for running tests (optional)

## Running the app

```bash
python app.py
```

## Running the tests

```bash
pip install pytest
pytest test_app.py -v
```
